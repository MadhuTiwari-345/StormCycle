## 🎯 Overview

**StormCycle** is a complete AI/ML femtech platform that transforms raw cycle data into:

- 🔍 Personalized cycle predictions for irregular cycles (26–50+ day ranges)
- 🩺 Clinical PCOD risk scoring using Bayesian adaptive screening
- 🤖 RAG-powered AI health chatbot with human-in-the-loop escalation
- 📄 One-click Doctor Report PDFs for gynecologist consultations
- 🔒 Medical-grade privacy with zero-knowledge analytics

## ⚙️ Key Design Decisions

### 🔹 LSTM over Rule-Based Prediction

* Captures temporal dependencies across multiple cycles
* Handles variable-length cycles (26–50+ days) natively
* No hardcoded thresholds → adapts to each user

---

### 🔹 Bayesian PCOD Screener

* Posterior probability updated with each new symptom signal
* Drastically reduces false positive rate vs. threshold-based tools
* Grounded in ACOG, NHS, and NIH clinical guidelines

---

### 🔹 zk-SNARK Privacy Architecture

* Pattern analysis performed on encrypted representations
* Raw health data never exposed to analytics pipeline
* Compliant with India's DPDP Act 2023 by design

---

## 📌 Assumptions

* Cycle data schema is flexible (supports 26–50+ day ranges)
* Missing cycle entries → flagged as potential clinical signal, not discarded
* Chatbot responses always cite clinical sources (ACOG · NHS · NIH)

## 🧾 Conclusion

StormCycle demonstrates that **menstrual health technology can be simultaneously personalized, clinically rigorous, and privacy-preserving**.

Through LSTM-based cycle modeling, Bayesian risk scoring, and RAG-powered clinical guidance, several structural advantages emerge:
